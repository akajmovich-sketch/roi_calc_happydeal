# ROI Calculator — HappyDeal

Одностраничный калькулятор ROI для AI-тренера продаж.

## Как посмотреть результат

1. Скачайте или клонируйте эту папку себе на компьютер.
2. Откройте файл `index.html` в браузере (можно просто перетащить его в окно Chrome/Edge/Safari).
3. Все расчёты выполняются на фронтенде, поэтому никакой сервер не нужен — страница работает сразу после открытия.
4. Если нужно выложить на GitHub Pages, в настройках репозитория выберите **Settings → Pages → Build and deployment → Deploy from branch**, затем укажите ветку `main` и папку `/docs`. GitHub подхватит готовый `docs/index.html` автоматически.
5. Если хотите поделиться ссылкой из локальной сети, можно запустить встроенный сервер:
   ```bash
   python3 -m http.server 8080
   ```
   После запуска откройте [http://localhost:8080/index.html](http://localhost:8080/index.html) в браузере.

## Что внутри

- **index.html** — готовый экран калькулятора с TailwindCSS и React.
- Все формулы расчёта описаны прямо в этом файле.

## Проверочные сценарии

Попробуйте в интерфейсе следующие комбинации, чтобы убедиться, что всё считает правильно:

- **Case A:** 20 / 3 / 120 000 / 2 мес / RUB → ROI должен быть положительным, появится срок окупаемости.
- **Case B:** 5 / 0 / 80 000 / 1 мес / RUB → эффект только от роста эффективности, ROI ниже.
- **Case C:** 35 / 4 / 2 500 / 3 мес / USD → цена берётся из USD-тиров, суммы в долларах.

## Дополнительно

Если хотите обновить расчёты или тексты, редактируйте константы в начале файла `index.html`. Комментарии в коде подскажут, какие формулы за что отвечают.
